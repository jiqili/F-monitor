# 展示监控数据图表的后台系统
建议使用yarn v3进行管理

## get start
```js
yarn 
yarn dev
```
## 命令解释
```js
//调试模式，每一次路由跳转都将从头渲染页面
yarn dev 
//生产模式，缓存页面内容进行ssr渲染
yarn build && yarn start
//如果需要修改端口直接使用 -p ${port}，如
yarn dev -p 3030
```
## 页面结构
```
src
├─components    //组件库
├─pages         //页面，内部的js文件约定为路由
└─utils         //工具库
```
## 图类型
1. 折线图：主要用来展示数据的趋势
2. 条形图：可以用来比对不同类型数据的数值
3. 饼图：可以直观展示不同数据的占比情况
